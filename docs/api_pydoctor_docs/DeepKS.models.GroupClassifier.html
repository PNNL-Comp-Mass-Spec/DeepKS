DeepKS.models.GroupClassifier

module documentation

Contains the abstract base class GroupClassifier for a group classifier. Also contains the (currently used) concrete subclass PseudoSiteGroupClassifier.

Class KinGroupClassifier Empty class denoting that the group classifier is grouping on kinases.
Class PseudoSiteGroupClassifier A group classifier that groups on the presence of a tyrosine at the center of the sequence. The predictions are, hence, either 'TK' or 'NON-TK'.
Class SiteGroupClassifier Empty class denoting that the group classifier is grouping on sites.
