class KinaseSubstrateRelationshipATTNWSELF(KSR):

View In Hierarchy

DeepKS model containing a torch.nn.MultiheadAttention layer for the site and kinase independently, which are used as a self-attention mechanism. These then get fed into non-self-torch.nn.MultiheadAttention layer after the CNN layer(s).
