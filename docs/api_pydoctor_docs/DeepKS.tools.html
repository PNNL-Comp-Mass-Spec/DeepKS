DeepKS.tools

package documentation

Contains miscellaneous utilities for DeepKS, including the main interface for training and evaluating (NNInterface).

Module bipartite_derangement Contains functions to generate derangements of bipartite sets, using maximum bipartite matching
Module custom_logging Module building on top of the logging module to provide a custom logger with custom logging functions and colors
Module custom_tqdm Implementation of a custom tqdm progress bar that works with logging.
Module custom_tree_maker Script to make a custom file tree for the DeepKS repo.
Module file_names Get a file name with a timestamp, with Windows/Onedrive name compatibility
Module formal_layers Contains custom layers for operations that don't exist as layers in PyTorch. Beneficial to see flow of data in the model summary.
Module get_array_percentile Functionality to get array of numbers depending on how each element numerically compares to axiswise percentiles
Module get_needle_pairwise Contains functionality to compute pairwise alignments using the needleall program from EMBOSS.
Module informative_tb Contains functionality to print a more informative traceback, giving the user a concrete format from which to debug.
Module legacy_plotting Old plotting functions that may be useful to take from in the future.
Module make_call_graph Make a visual call graph to help understand the path of the code
Module make_call_graph_demonstrator Simple demonstrator/example of how to use make_call_graph.py
Module make_fasta Make a fasta file from a dataframe with biological sequences.
Module model_utils Contains various useful functions for working with models (specifically CNNs) and data
Module nice_printer Contains custom pprint implementations.
Module NNInterface Contains NNInterface, A flexible class that allows for training, validation, testing, and summarization of a torch.nn.Module-based neural network
Module raw_score_to_prob Module to help convert raw classifier scores into probabilities.
Module roc_helpers Helper functions for computing ROC curves and AUC scores
Module schema_validation Validation schemas for Kinase and Site metadata inputs.
Package splash Used to write splash screens to the console, to confirm the program that is running.
Module system_tools Utilities that interact with the system, e.g. run shell commands and get output.
Module tensorize Functions to convert textual biological sequences into tensors for deep learning models.
Module Tuner Module to perform hyperparameter tuning on torch.nn.Module -based neural networks.
