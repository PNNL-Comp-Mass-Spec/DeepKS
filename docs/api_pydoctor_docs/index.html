Package containing tool to investigate and predict kinase-substrate interactions.

Package api The api package allows one to use DeepKS to make true pair/false pair predictions, based on input kinases and sites. Essentially, the main submodule is an input-validated wrapper for models.MultiStageClassifier...
Package config Contains overarching configuration options for DeepKS.
Package data Contains scripts to gather, preprocess, and format all data for DeepKS.
Package discovery Contains scripts to use DeepKS with novel data/applications (e.g., validation with Nature Atlas)
Package examples Module containing example scripts
Package images Module containing scripts to generate images/plots for DeepKS. The images/plot output fils are also stored here.
Package models Contains Group Classifier and Neural Network definitions and specific training and predicting functionality.
Package tests Contains all test-related files for DeepKS.
Package tools Contains miscellaneous utilities for DeepKS, including the main interface for training and evaluating (NNInterface).
